# 11. Website Structure and Brand Messaging

## Information Architecture

- Homepage
- Product
- Use Cases
- Security & Trust Center
- Pricing
- Docs and Resources
- Contact / Demo

## Messaging Style

Theia should be positioned as:
- A trusted orchestration and observability layer
- Consent-based and privacy-first
- Explainable and audit-ready

Avoid:
- Hype language
- "Guaranteed correctness" claims
- Surveillance-like framing

## Core Messaging

- "Operate AI agents with evidence, oversight, and control."
- "Understand memory, workflows, and reasoning checkpoints through a transparent, permission-based control center."
- "Scale agent operations without sacrificing trust."
- "Free public rollout with transparent upgrade policy."
