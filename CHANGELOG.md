# Changelog

## [0.1.0] - 2026-04-25

### Added
- Monorepo foundation for Theia platform
- Desktop dashboard shell with core operational views
- Local core service with ingestion/timeline/alert endpoints
- Optional control-plane service with SAML-ready auth and local login-volume dashboard
- Canonical event schema package
- Connector SDK with local file connector, Codex CLI connector, and custom JSON connector
- Reasoning quality detection package
- Policy/permission engine with audit chaining
- Product, architecture, trust, and GTM documentation (sections 1-13)
- Website messaging and trust-center copy pack
- GitHub governance docs and CI scaffolding
