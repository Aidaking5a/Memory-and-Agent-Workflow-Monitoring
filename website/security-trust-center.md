# Security and Trust Center

## Security Posture

- Least privilege by default
- Consent-based connector access
- Scope-bound file and event permissions
- Immutable audit chain for critical actions

## Privacy Model

- Local-first processing by default
- No covert collection or stealth monitoring
- Optional cloud sync with explicit enablement
- Redaction controls for exports

## Responsible Use

Theia is designed for authorized observability and memory orchestration only. It does not support unauthorized access, stealth persistence, or offensive automation patterns.
