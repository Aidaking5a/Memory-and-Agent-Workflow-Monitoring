> Quick Start

A clean install path for customers who want the command center running locally before they start pairing OpenClaw, Octopoda, MCP, API, OAuth, or terminal agents.

Configure

Downloads the Windows installer script, prints it for review, builds the dashboard, and starts Theia.

$ powershell -NoProfile -ExecutionPolicy Bypass -Command "$u='https://raw.githubusercontent.com/aidaking5a/Memory-and-Agent-Workflow-Monitoring/main/scripts/install-theia-command-center.ps1'; $p=Join-Path $env:TEMP 'install-theia-command-center.ps1'; Invoke-WebRequest $u -OutFile $p; Get-Content $p; powershell -NoProfile -ExecutionPolicy Bypass -File $p -BuildDashboard -StartAfterInstall"
Prints script before running No silent Docker or WSL No paid API setup Reversible local install

The Customer Interface Is The Product.

Theia reduces the control center to three core experiences: stats, the agent card deck, and the live reporting infographic with bubble activity, explicit links, and safe controls.

Agent Stats

Computing power, token usage, RAM, runtime, model usage, API usage, spend, and per-agent breakdowns.

FIFA-Style Agent Cards

Model, vendor, memory summary, skills, connectors, usage, status, and trust level in a quick card deck.

Live Network Infographic

Agent bubbles scale by activity, links show collaboration, and the operator can query, steer, link, break, or stop agents.
