# Trust Center Draft

## What Theia Accesses

Only user-authorized connectors, files, and event scopes.

## What Stays Local

Parsing, normalization, timeline building, and alert analysis are local-first by default.

## Optional Remote Features

Cloud sync and team governance controls can be enabled explicitly by workspace owners.

## How to Revoke Access

Permissions are visible and revocable in the dashboard. Revocations are immediate and audit-logged.

## Data Deletion

Workspace owners can request local purge and remote data deletion workflows according to plan tier.
