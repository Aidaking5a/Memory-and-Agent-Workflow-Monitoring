# Product Page

## Product Overview

Theia gives operators a transparent, permission-based view of agent workflows, memory revisions, and reasoning checkpoints.

## Core Modules

- Overview Dashboard
- Agent Health
- Workflow Timeline
- Memory Explorer
- Reasoning Alert Center
- Agent Comparison
- Audit and Permissions
- Settings and Connector Management
- Optional control-plane login observability dashboard

## Why Teams Choose Theia

- Faster diagnosis of workflow failures
- Better memory hygiene and context reliability
- Lower operational risk through explainable oversight
- Governance-ready controls without surveillance patterns
