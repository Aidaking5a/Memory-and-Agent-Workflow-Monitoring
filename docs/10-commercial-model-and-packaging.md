# 10. Commercial Model and Packaging

## Current Commercial Phase

Theia is currently launching as **Free** to maximize adoption, trust, and product feedback quality.

## Free Plan Scope (Now)

- Local-first workflow observability
- Memory file ingestion (`memory.md`, `bootstrap.md`)
- Codex CLI log connector
- Custom JSON log connector
- Reasoning alert baseline
- Optional local control-plane with SAML-ready auth and login-volume dashboard

## Why Free First

- Rapid design-partner acquisition
- Faster iteration on alert precision and UX
- Higher visibility before monetization
- Lower trust friction while proving value

## Future Packaging Intent (Deferred)

Paid tiers are intentionally deferred until product-market fit and trust posture are validated at scale.

Future restrictions will be introduced transparently with clear value boundaries and no hidden data monetization.
