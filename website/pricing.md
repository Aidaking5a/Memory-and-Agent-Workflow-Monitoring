# Pricing

## Current Availability

Theia is currently **Free** while we scale design-partner adoption and refine reasoning-quality workflows.

## Included in Free

- Local-first desktop observability
- Codex CLI and custom JSON connectors
- Memory orchestration timeline
- Reasoning alert center
- Optional local control plane with SAML-ready login and login-volume dashboard

## Commercial Transition Policy

Paid packaging will be introduced later with transparent release notes, clear value boundaries, and no hidden data monetization.
